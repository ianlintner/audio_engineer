# Claude Agent Setup

This folder is reserved for Claude agent-specific customizations.

Suggested structure:
- `.claude/agents/<agent-name>.md` for role-specific instructions.
- `.claude/skills/<skill-name>/SKILL.md` for reusable task guidance.

Example use cases:
- `midi-style-refiner`: style-aware pattern tuning.
- `theory-validator`: enforces harmony/rhythm invariants.
- `api-contract-keeper`: validates API schema compatibility.
